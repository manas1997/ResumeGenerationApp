# Roadmap

## Phase 1: MVP

- Resume upload and parsing.
- JD parsing.
- Source claim extraction.
- Grounded tailoring.
- Scorecard.
- PDF export.
- Source-PDF template snapshot extraction.
- Template-preserving PDF export with 99% fidelity gate.
- In-browser PDF preview.
- Download actions for original, tailored, and ATS report PDFs.
- ATS dashboard and skill improvement report.
- Version history.

## Phase 2: Review Workflow

- Inline editing.
- Accept/reject bullet rewrites.
- Unsupported keyword clarification prompts.
- DOCX export.
- Multiple templates.
- More robust embedded-font replacement support.
- Visual diff thumbnails for original vs generated PDF.

## Phase 3: Intelligence Improvements

- Embedding-based semantic ranking.
- Skill taxonomy and alias management.
- Resume benchmark analytics.
- Role-family templates.

## Phase 4: Scale

- Organization accounts.
- Team templates.
- Bulk tailoring.
- Human review marketplace hooks.
- Multi-region deployment.
